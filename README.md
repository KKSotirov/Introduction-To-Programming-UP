## 📂 Специализирани секции в репозиторито

Освен седмичните упражнения, проектът съдържа и допълнителни папки, разпределени по предназначение:

* **`УП homework` / `УП ПРАКТИКУМ ДР`** – Официални домашни работи и проекти, оценявани по време на семестъра.
* **`Семинари`** – Допълнителен код, писан по време на редовните семинарни занятия.
* **`PREP FOR TEST 1` / `UP pract Test1`** – Изпитни задачи, симулации на тестове и подготовка за Контролно №1 и Контролно №2.
* **`Random programs`** – Мои лични експерименти, бележки и тестване на парчета код извън задължителната програма.
