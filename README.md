# Увод в програмирането (УП) 💻

Това репозитори съдържа архив със задачи, лабораторни упражнения и домашни по курса **Увод в програмирането** във ФМИ.

## 📂 Структура на проекта
 Проектът е организиран тематично по папки спрямо изучавания материал в университета.

## 🛠️ Използвани технологии
* **Език за програмиране:** C++
* **Среда за разработка:** Visual Studio Code (VS Code)
* **Версионен контрол:** Git / GitHub
